---
description: Push project to GitHub
---

1. Initialize Git repository
2. Add all files
3. Commit changes
4. Rename branch to main
5. Add remote origin
6. Push to origin

```bash
git init
git add .
git commit -m "Complete project implementation: Backend logic, API client, and Build fixes"
git branch -M main
git remote add origin https://github.com/badpower45/dd.git
git push -u origin main
```
